dog.h

Header file containing the definition of struct dog and the typedef dog_t, along with function prototypes.

1-init_dog.c

Function to initialize a variable of type struct dog.

2-print_dog.c

Function to print the contents of a struct dog, handling NULL strings/pointers.

4-new_dog.c

Function to create a new dog_t dynamically, allocating memory for and copying the name and owner strings.

5-free_dog.c

Function to properly free the memory allocated by new_dog.
