0x15. C - File I/O

This project focuses on the fundamental concepts of file Input/Output (I/O) in C, utilizing low-level system calls as required by the project constraints. The system calls used include open, read, write, and close.

Tasks

File

Description

0-read_textfile.c

Function to read a text file and print its contents to the POSIX standard output.

1-create_file.c

Function to create a file with specific permissions (rw-------) and write a given text content. Overwrites and truncates if the file exists.

2-append_text_to_file.c

Function to append text content to the end of an existing file.

3-cp.c

A program that copies the content of one file (file_from) to another (file_to), implementing robust error handling and using a 1024-byte buffer.
